## 今後の実装計画

### Phase 7: パーティクルシステム
- **ParticleEmitter コンポーネント**: エンティティからパーティクルを発射
- **パラメータ**: 発射レート、寿命、初速度、重力影響、サイズカーブ
- **ビジュアル**: テクスチャ、色グラデーション、ブレンドモード
- **スクリプトAPI**: `particles.emit()`, `particles.stop()` など
- **Inspector UI**: パーティクルパラメータのリアルタイム編集
- **プリセット**: 炎、煙、爆発、キラキラなどのテンプレート

### Phase 8: アニメーションシステム
- **AnimationClip**: キーフレームベースのアニメーション定義
- **AnimationPlayer コンポーネント**: クリップの再生・ブレンド
- **タイムラインエディタ**: GUI でキーフレームを打つパネル
- **プロパティアニメーション**: position, rotation, scale, color 等のアニメート
- **Tween API**: スクリプトからのイージングアニメーション
- **GLB アニメーション**: インポートしたモデルのアニメーション再生

### Phase 9: シーン管理＆永続化
- **シーンの保存/読み込み**: localStorage / IndexedDB への永続化
- **複数シーン**: シーン切り替え機能
- **プレハブシステム**: エンティティテンプレートの保存・再利用
- **Undo/Redo**: 操作履歴のスタック管理
- **自動保存**: 一定間隔での自動保存機能

### Phase 10: レンダリング強化
- **ポストプロセス**: Bloom, SSAO, Vignette, Color Grading
- **シャドウ改善**: PCF ソフトシャドウ、カスケードシャドウマップ
- **環境マップ**: HDRI スカイボックス / 反射プローブ
- **テクスチャマッピング**: ディフューズ、ノーマル、ラフネスマップのサポート
- **GPUインスタンシング**: 大量オブジェクトの効率的レンダリング

### Phase 11: 2Dゲームサポート
- **2D モード**: Orthographic カメラ、2Dビュー切り替え
- **SpriteRenderer**: 2D スプライト描画コンポーネント
- **スプライトシート**: アニメーションスプライトのサポート
- **2D物理**: Rapier の 2D コリジョン対応
- **タイルマップ**: グリッドベースのレベルエディタ

### Phase 12: ネットワーキング＆マルチプレイヤー
- **WebSocket / WebRTC**: リアルタイム通信基盤
- **ネットワーク同期**: エンティティの位置同期
- **ロビーシステム**: ルーム作成・参加
- **スクリプトAPI**: `network.send()`, `network.onMessage()` など

## 開発環境セットアップ

```bash
# 依存関係のインストール
npm install

# 開発サーバー起動
npm run dev

# ビルド
npm run build
```

ブラウザで `http://localhost:5173/` を開くとエディタが起動します。
